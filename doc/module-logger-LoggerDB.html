Class: LoggerDB

logger~LoggerDB()

new LoggerDB()

.message() - Adds message in order to save to DB

.done() - Converts all to string and saves it to localDB with OK flag

.error() - Ignores messages and adds error string. It is saved to localDB with ERROR flag
